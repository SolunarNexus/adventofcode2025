# Advent of Code 2025

Solutions to the [Advent of Code 2025](http://adventofcode.com/2025) set of problems using the Kotlin programming language.

These folutions are #AIFree, i.e. solved without AI augmentation. I treat these exercies as a place to learn something new about programming languages as well as problem solving (for these nasty interviews).
